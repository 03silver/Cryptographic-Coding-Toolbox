# CommonCryptographicEncodingConversions(常见加密编码转换)
**You can choose a common cryptographic encoding to convert to each other**

---

## Cryptographic encoding is available
+ ### base16
+ ### base32
+ ### base64
+ ### base85
+ ### BIN
+ ### HEX

## Features
+ ### AutoCopy
+ ### ReverseMode

## languages
+ ### zh-cn
+ ### zh-hk
